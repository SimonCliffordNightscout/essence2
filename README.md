## How to use
1. Login to your WonderCMS website.
2. Click "Settings" and click "Themes".
3. Scroll to the bottom, enter https://raw.githubusercontent.com/SimonCliffordNightscout/essence2/master/wcms-modules.json to +ADD
4. In the "General" tab, select theme to activate it.
